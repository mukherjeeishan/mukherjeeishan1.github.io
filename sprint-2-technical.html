Margins Borders and Padding


When we start to design and add styles to our website using css we will often hear the words margin, padding and border thrown around a lot. It is important that we have a good understanding of how these elements work as they are a key part of making our websites look good and professional.

Padding refers to the space inside an element between the content inside the element and the border of the element. Padding goes around all four sides of the element and we can target and change padding for individual sides using CSS.

Margin is the space between the border and the next element. Just like padding it goes around all four sides of the element and we can target and change individual sides using CSS to change the margins.
